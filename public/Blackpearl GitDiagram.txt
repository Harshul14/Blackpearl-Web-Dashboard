  %% Client flows
  React -->|uses| UI
  React -->|uses| Hooks
  React -->|uses| TRPCReact
  React -->|uses| QueryClient
  TRPCReact -->|calls| API_tRPCRoute
  QueryClient -->|calls| API_tRPCRoute

  %% API routing
  API_tRPCRoute -->|routes to| TRPCServer
  TRPCServer -->|imports| TRPCRoot
  TRPCRoot -->|includes| Routers
  Routers -->|queries| DBClient
  DBClient -->|defined by| DBSchema

  React -->|SSR data fetch| DBClient

  %% Billing flow
  React -->|POST /api/webhook/stripe| API_Stripe
  API_Stripe -->|updates| DBClient
  API_Stripe -->|notifies| Stripe

  %% Meeting processing
  React -->|POST /api/process-meeting| API_Process
  API_Process -->|invokes| Gemini
  Gemini -->|returns results| API_Process
  API_Process -->|writes| DBClient

  %% GitHub integration
  TRPCRoot -->|calls| GitHubAPI
  GitHubAPI -->|returns metadata| Routers

  %% Auth middleware
  React -->|edge auth| Middleware
  Middleware -->|protects| API_tRPCRoute
